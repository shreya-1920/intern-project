# frontend-intern-assignment
MINI PRODUCT DASHBOARD 
This is a simple PRODUCT DASHBOARD built using html , css and javascript.
it fetches data from FakeStore API , displays products , searches the items , filters , sorting , theme toggle button and favourites.

FEATURES:
1. Fetch products from API
2. Search product by title
3. Sort by price
4. Dark/light theme toggle
5. Add to favourites
6. Responsive design
7. Product details modal
8. Filter by category

TECH USED:
HTML
CSS
JAVASCRIPT
FAKESTORE API


FOLDER STRUCTURE:
/project folder
|----index.html
|----style.css
|----script.js

How to Run the Project
1.Download or clone the repo
2.Open index.html in your browser
3.Done
